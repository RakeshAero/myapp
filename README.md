# ToDo List

ToDo List using Laravel 


## Requirements



```bash
  php (8.0.5)
  composer (2.8.1)
```

## Installation

Open the Project folder `myapp`

Open Terminal and type the below commands


```bash
  1.git clone https://github.com/RakeshAero/myapp.git
  2. composer install
  3. php artisan serve
```
